# chromecredentials

## Description

This python application prints a table containing all chrome creadentials from your OSX or Linux machine.

| website  |    username   |  password |
|----------|:-------------:|----------:|
| facebook |   username    |  password |
|   gmail  |   username    |  password |
|   .....  |    ......     |   ......  |
